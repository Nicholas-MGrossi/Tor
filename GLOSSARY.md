# Glossary (XPII.Net)

## Tor
A *temporary* repository name used during setup on GitHub. It is not related to (and not affiliated with) the Tor anonymity network.

## UserDev
A placeholder term for the operator’s local development workspace used while setting up the repository. It is not required by the project; the authoritative source is the GitHub repository.

## warp-agent
The automated assistant/agent environment used to help perform repository setup tasks (e.g., creating files, running git commands) within Warp.

## GitHub Pages
GitHub’s static hosting for the `docs/` site content.

## gitleaks
An open-source secret scanning tool used in CI to detect accidental commits of secrets.

## Dependabot
GitHub’s automation for proposing dependency updates.

# XPII.Net

This repository is intended for publication-ready materials.

## Security & privacy baseline
- Do not commit secrets (API keys, tokens, private keys, credentials, personal data).
- Review changes via pull requests.
- Prefer least-privilege access.

## Quick start
- Put published materials in `docs/`.
- Update `README.md` with purpose, scope, and contact info.

## Reporting security issues
See `SECURITY.md`.

Architecture Overview

What this repository is

XPII.Net is a publication-focused repository with a static documentation site deployed from docs/ to GitHub Pages.

Components

  • Static site: files in docs/ are deployed as-is.
  • CI secret scanning: a workflow runs gitleaks on pushes and pull requests.
  • Automation: Dependabot updates GitHub Actions dependencies weekly.

Security model (baseline)

  • Assume public visibility; do not publish credentials or private data.
  • Use pull requests + review as the primary safety gate.
  • Automated scanning helps reduce mistakes, but it is not perfect.

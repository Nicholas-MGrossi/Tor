Glossary

Tor

A temporary GitHub repository name used during setup. Not related to (and not affiliated with) the Tor anonymity network.

UserDev

A placeholder term for the operator’s local development workspace used while setting up the repository.

warp-agent

The automated assistant/agent environment used to help perform repository setup tasks within Warp.

GitHub Pages

Static hosting that publishes the contents of docs/.

gitleaks

Secret scanning tool used in CI.

Dependabot

Automation for dependency update pull requests.

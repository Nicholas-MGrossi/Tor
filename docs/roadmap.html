Project Goals & Roadmap

Goals

  • Provide a clean, public set of documents suitable for publication.
  • Maintain a strong baseline against accidental data disclosure.
  • Keep deployment simple, reproducible, and auditable.

Immediate tasks

  • Enable GitHub Pages (Settings → Pages → Source: GitHub Actions).
  • Set repository visibility to Public (if intended).
  • Set branch protection for main (recommended): PRs + required checks.
  • Fill in a real private contact method in SECURITY.md.

Near-term hardening

  • Add link checking for docs pages.
  • Add content review policy and change control rules.
  • Optional: add a static-site generator if you want richer navigation/search.
